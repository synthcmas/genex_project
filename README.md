# genex_project

This repository contains the interactive `run_genex.ipynb` notebook that wraps the Fortran `genex` simulation. 

## Running interactively via Binder

You can launch the interactive notebook without installing anything locally:

[![Binder](https://mybinder.org/badge_logo.svg)](https://hub.gesis.mybinder.org/user/synthcmas-genex_project-t5asif6v/doc/tree/run_genex.ipynb)

## Project Structure

- `run_genex.ipynb` : Main interactive notebook.
- `src/genex_wrapper.py` : Python wrapper for running the Fortran simulation.
- `fortran/` : Fortran source files and Makefile.
- `data/` : Input data files required by the notebook.
- `plot_data/` : Output files generated by `genex`.
- `figs/` : Generated plots.
- `requirements.txt` : Python dependencies.
- `postBuild` : Binder postBuild script to enable ipywidgets.

## Notes

- The notebook is interactive using `ipywidgets`.
- Ensure `data/` and `plot_data/` are present for the notebook to run correctly.
- Python plotting uses `matplotlib`; seaborn is optional.
